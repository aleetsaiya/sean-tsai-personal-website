# Personal Website

A clean, minimalist personal portfolio website built with [Next.js](https://nextjs.org) and [SCSS](https://sass-lang.com/).

## Sections

- **About** — A brief introduction about myself
- **Experience** — Work experience with skills and links
- **Projects** — Highlighted personal and professional projects

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: SCSS Modules
- **Font**: Roboto (via `next/font`)

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Content

All site content (name, about, experience, projects) is managed through a single [`setting.json`](./setting.json) file.
